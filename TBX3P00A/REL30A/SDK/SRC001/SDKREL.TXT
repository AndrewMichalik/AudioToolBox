                Audio ToolBox Development Kit & Programming Samples
                               Version  3.00
                                  10/01/96


3.00a.1:
    Added SDK support for all ToolBox functions.
    Added additional .h and .txt definitions
    Implemented PCMCvt ulDstLen parameter.
    Check for NULL callbacks in all tools.
    Increased size of function block, added initialization functions.
    Added sample "C" programs for all SDK functions.
    Source and destination frequencies now PCM context sensitive.
    SndChp supports all native PCM types.
    Improved all filter characteristics.
    Improved CVSD fidelity.
    Improved clipping on all volume adjust operations.
    Improved speed on 386-class machines when DCE is inhibited.
    Improved Equalizer for all memory conditions.
    Improved DTMF filter. 
    Improved IIR filter.
    Volume normalize default changed to .2% over range.
    Added -q6 anti-aliasing filter.
    SndChp: Accepts destination directory argument like "\temp\.vox"    
    Fixed header size error on certain wave files.
    Improved support for extended wave file formats.
